# 🤖 PawPal ML Models

This directory contains machine learning models for pet breed classification.

## 📁 Structure

```
ML_Models/
├── cats/
│   ├── data/          # Cat breed dataset
│   └── notebooks/     # Jupyter notebooks for cat breed classification
└── dogs/
    ├── data/          # Dog breed dataset  
    └── notebooks/     # Jupyter notebooks for dog breed classification
```

## 🎯 Models

- **Cat Breed Classification**: Using Oxford-IIIT Pet Dataset
- **Dog Breed Classification**: Using Stanford Dogs Dataset

## 🚀 Getting Started

1. Download datasets into respective `data/` folders
2. Open Jupyter notebooks in the `notebooks/` folders
3. Follow the training pipeline in each notebook
